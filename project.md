# Навигатор самочувствия

Telegram Mini App для нутрициологов, wellness-консультантов и специалистов по здоровью. Помогает человеку за 3 шага понять, что стоит за его симптомами, и мягко подводит к консультации со специалистом.

**Живая ссылка:** https://mvpextracted.vercel.app
**Telegram бот:** @my_health_navigator_bot
**GitHub:** https://github.com/valerairon70-hub/health-navigator-miniapp

---

## Для кого

- **Пользователь:** человек с размытыми жалобами (усталость, плохой сон, лишний вес, стресс, ЖКТ)
- **Владелец:** wellness-консультант, который получает прогретых лидов через бота

---

## Проблема — зачем это нужно

Wellness-специалист теряет потенциальных клиентов, потому что человек с жалобами не понимает, зачем ему консультация — он не осознаёт связь между симптомами и причинами. Стандартный сценарий: "напиши мне в личку" → человек не пишет, потому что не знает с чего начать.

Навигатор решает это: за 3 шага он объясняет человеку его симптомы его же языком, формирует доверие и создаёт естественный запрос на разговор со специалистом.

---

## Бизнес-модель

Приложение работает как инструмент прогрева: пользователь проходит навигатор → получает объяснение своих симптомов → видит карточку специалиста → нажимает кнопку и попадает в Telegram-чат с предзаполненным сообщением о своей проблеме. Дальше специалист ведёт разговор вручную и закрывает на консультацию. Само приложение бесплатное — монетизация на стороне специалиста.

---

## Основные функции

- Выбор симптома из 6 тем (усталость, сон, ЖКТ, вес, стресс, "не знаю с чего начать")
- Объяснение причин симптома простым языком без медицинских терминов
- Выбор формата следующего шага (разбор / трекер / вопрос)
- Карточка специалиста с социальным доказательством и счётчиком мест
- Прямой переход в Telegram-чат со специалистом с предзаполненным сообщением

---

## Чего НЕ будет (пока)

- Постановки диагнозов или медицинских рекомендаций — это не медицинский сервис
- Регистрации, личного кабинета, хранения данных пользователя
- Чат-бота с ответами — только живой специалист
- Нескольких специалистов / маркетплейса

## Экраны (текущие)

| Экран | Описание |
|-------|----------|
| `intro` | Заголовок, оффер, список выгод, кнопка старта |
| `welcome` | Выбор темы (6 карточек симптомов) |
| `explanation` | Объяснение темы + 4 зоны внимания |
| `selection` | Выбор формата следующего шага (3 варианта) |
| `action` | Карточка специалиста + финальный CTA → открывает чат в Telegram |

## Стек и дизайн

**Технологии:** React 19 · TypeScript · Vite · Tailwind CSS 4 · Motion (анимации) · Lucide React (иконки)
**Деплой:** Vercel (автодеплой из GitHub)
**Интеграция:** Telegram WebApp SDK (`tg.ready`, `tg.expand`, `tg.openTelegramLink`)

**Дизайн-система:**
- Фон: `#f5f5f0` (тёплый офф-вайт)
- Акцент: `#5A5A40` (тауп / оливковый)
- Шрифт заголовков: serif (Georgia-style)
- Шрифт текста: system sans-serif
- Анимации: fade + slide при переходе между экранами

**Маркетинг:** принципы Dan Kennedy — специфика, срочность, социальное доказательство, бесплатный оффер, конкретный CTA.
