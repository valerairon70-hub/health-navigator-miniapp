# Навигатор самочувствия

Telegram Mini App для wellness-специалистов и MLM-предпринимателей в сфере здоровья. Помогает человеку за 3 шага понять, что стоит за его симптомами, и мягко подводит к консультации со специалистом.

**Живая ссылка:** https://mvpextracted.vercel.app
**Telegram бот:** @my_health_navigator_bot
**GitHub:** https://github.com/valerairon70-hub/health-navigator-miniapp

---

## Для кого

- Пользователь: человек с размытыми жалобами (усталость, плохой сон, лишний вес, стресс, ЖКТ)
- Владелец: wellness-консультант, который получает прогретых лидов через бота

---

## Экраны (текущие)

| Экран | Описание |
|-------|----------|
| `intro` | Заголовок, оффер, список выгод, кнопка старта |
| `welcome` | Выбор темы (6 карточек симптомов) |
| `explanation` | Объяснение темы + 4 зоны внимания |
| `selection` | Выбор формата следующего шага (3 варианта) |
| `action` | Карточка специалиста + финальный CTA → открывает чат в Telegram |

## Стек и дизайн

**Технологии:** React 19 · TypeScript · Vite · Tailwind CSS 4 · Motion (анимации) · Lucide React (иконки)
**Деплой:** Vercel (автодеплой из GitHub)
**Интеграция:** Telegram WebApp SDK (`tg.ready`, `tg.expand`, `tg.openTelegramLink`)

**Дизайн-система:**
- Фон: `#f5f5f0` (тёплый офф-вайт)
- Акцент: `#5A5A40` (тауп / оливковый)
- Шрифт заголовков: serif (Georgia-style)
- Шрифт текста: system sans-serif
- Анимации: fade + slide при переходе между экранами

**Маркетинг:** принципы Dan Kennedy — специфика, срочность, социальное доказательство, бесплатный оффер, конкретный CTA.
