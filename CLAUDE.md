# Правила работы с проектом

## Общее
- Это Telegram Mini App — все решения принимаем с учётом мобильного формата внутри Telegram
- Перед созданием нового файла — сначала объясни, что планируешь сделать и зачем

## Как запустить локально

```bash
cd mvp_extracted
npm install
npm run dev
```

Приложение запускается на `http://localhost:3000`

Перед запуском создай файл `.env` на основе `.env.example`:
```
GEMINI_API_KEY=your_key_here  # в текущем UI не используется, зарезервировано на будущее
APP_URL=http://localhost:3000
```

Для деплоя: пушим в `main` → Vercel деплоит автоматически.
