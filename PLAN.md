## 🔴 Приоритет 1 — Конверсия (сделать сейчас)

### Скрипт первого ответа специалиста
**Без этого воронка обрывается.** Человек написал — тишина — ушёл.
Нужен готовый шаблон ответа в первые 2 часа:
```
Привет! Я Валерий, получил твой запрос через Навигатор.
Ты выбрал(а) тему: [симптом].
Давай начнём с короткого созвона/переписки — расскажи подробнее...
```
- [ ] Написать 6 версий скрипта под каждый симптом (energy, sleep, digestion, weight, stress, other)
- [ ] Сохранить в отдельный файл `scripts/specialist-reply.md`

### Реальные данные в соцдоказательствах
Фейковые цифры подрывают доверие — особенно в wellness-сообществе, где всё проверяется:
- [ ] `count: 340` — заменить реальной цифрой или убрать до накопления данных
- [ ] `SPECIALIST.fact: '200+ клиентов'` — подтвердить или скорректировать
- [ ] `SPECIALIST.testimonial` — заменить «Марину, 34» реальным отзывом как только появится первый клиент

### Слоты на неделю
- [ ] `slotsLeft: 3` — обновить на реальное значение перед запуском кампании, либо убрать блок совсем (если реального дефицита нет — не создавать ложного)

## Метрика успеха

> «Работает» = человек дошёл до CTA и нажал кнопку, Валерий получил сообщение.

| Метрика | Цель MVP | Цель через 3 мес |
|---|---|---|
| Intro → Action (доходимость) | > 60% | > 75% |
| Action → CTA click | > 50% | > 65% |
| CTA → реальный диалог | > 70% | > 80% |
| Диалог → консультация | измерить вручную | > 30% |
